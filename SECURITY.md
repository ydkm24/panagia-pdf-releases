# Security

## Reporting Security Issues

Please report security issues privately to the project owner. Do not open public issues containing credentials, tokens, or exploit details.

## Downloads

Only download installers from the official GitHub Releases page for this repository. Verify the SHA256 checksum before running an installer when possible.

## Secrets

This release repository must not contain application source secrets, `.env` files, API keys, personal access tokens, Supabase service keys, Stripe keys, or user session files.
