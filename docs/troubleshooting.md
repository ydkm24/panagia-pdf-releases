# Troubleshooting

## App does not launch

- Restart Windows and try again.
- Ensure Microsoft Edge WebView2 Runtime is installed.
- Reinstall using the latest installer.

## Login or cloud features fail

- Check internet connection.
- Restart the app.
- Confirm you are using the latest release.

## Installer will not replace an old copy

- Close Panagia.pdf.
- Check Task Manager for `Panagia.exe` and end it if needed.
- Run the installer again.
