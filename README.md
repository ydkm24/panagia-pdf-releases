# Panagia.pdf Releases

Native Windows desktop installer downloads for **Panagia.pdf**.

## Download

Latest installer will be attached to the newest GitHub Release:

- **Latest release:** `v0.1.0`
- **Installer:** `Panagia.pdf_Setup.exe`

> The installer binary is uploaded as a GitHub Release asset. It is not committed into this repository.

## Requirements

- Windows 10 or newer
- Microsoft Edge WebView2 Runtime
- Internet connection for login, cloud sync, publishing, and AI translation features

## Install

1. Download `Panagia.pdf_Setup.exe` from the latest release.
2. Run the installer.
3. Launch Panagia.pdf from the Start Menu or Desktop shortcut.

## Verify Download

Compare the installer SHA256 with the checksum in `checksums/`.

